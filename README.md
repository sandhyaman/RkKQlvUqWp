# 智能旅游推荐系统 python1174

## 项目概述

智能旅游推荐系统是一个基于Python语言的综合性项目，融合了Django框架和Vue.js技术栈，致力于为用户提供优质的旅游信息推荐与交流平台。

## 技术栈

- **后端：**
  - Python 3.7
  - Django框架
- **前端：**
  - Vue.js
- **数据库：**
  - MySQL
- **开发工具：**
  - PyCharm

## 功能模块

- **首页**：展示旅游推荐及热门资讯
- **个人中心**：用户资料、行程管理等
- **用户管理**：用户注册、登录、权限管理等
- **旅游资讯管理**：旅游新闻发布、更新等
- **景点信息管理**：景点详情展示、分类管理等
- **酒店信息管理**：酒店信息展示、预订管理等
- **行程分享管理**：用户分享行程经验
- **交流论坛**：用户互动交流区
- **系统管理**：系统设置、权限分配等

## 系统角色

- 游客
- 注册用户
- 管理员

## 运行环境

- Python 3.7+
- Django 2.x
- Vue.js 2.x
- MySQL 5.7+

## 部署步骤

1. 环境准备：安装Python 3.7+、MySQL数据库、Node.js等
2. 克隆代码到本地
3. 安装依赖包：使用`pip`安装Python依赖；使用`npm`安装前端依赖
4. 数据库配置：根据项目文档设置数据库参数
5. 运行后端服务：使用Django命令运行项目
6. 前端编译与部署：编译Vue项目并部署到Web服务器

## 目录结构

```
project/
│
├── backend/          # 后端Django项目目录
├── frontend/         # 前端Vue项目目录
└── database/         # 数据库文件
```

## 核心亮点

- **个性化推荐**：基于用户行为和偏好，智能推荐旅游行程
- **高度可定制**：前端使用Vue.js，界面响应式设计，易于维护和扩展
- **高效开发**：后端采用Django框架，提升开发效率
